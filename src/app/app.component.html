Visual Power at Scale?

Fuse D3 and Angular 2 in TypeScript


Perfect Storm

Ever wondered, how to build modern web apps with re-usable, interactive data visualization components?

The latest incarnation of D3 is getting you excited, Angular 2 is approaching the landing strip, and you have come to love TypeScript for its promise to develop complex solutions at scale.

But integrating all three seems daunting.

Take Flight

This Angular 2 demo app and its source code repository are intended to get you started.

They illustrate how D3 version 4 visualizations can be implemented as Angular 2 components in TypeScript 2.

Rather than trying to be exhaustive or writing a lengthy blog post, a select few well-known D3 demo scripts have been adapted to highlight key considerations.
