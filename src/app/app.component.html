Background

This Angular 2 App is intended as a very short demo illustrating the use of a D3 Service in conjunction with Angular 2 components.

The visualizations below are adaptations of popular examples provided by Mike Bostock. They are simply intended to show how D3 version 4 can be used when building Angular 2 components. The implementations have been left close to the original to allow cross-referencing as a learning aid. However, changes naturally arise due to:

  • the structure of Angular 2 components,
  • the use of TypeScript, and
  • the migration of some of the examples to D3 version 4.

The complete source code of this demo can be found on Github at tomwanzek/d3-ng2-demo.
